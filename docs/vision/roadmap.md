# Roadmap

## Phase 1 (current)

Join messages, scheduled restart, `/omv` admin commands, verify gates.

## Phase 2

MOTD and tab-list customization, maintenance mode, persisted first-join data.

## Phase 3

Cross-server announcements, server selector UX, lkjmcsmp message parity.

## Phase 4

Rate limiting, staff tools, metrics hooks.

## Phase 5

Docker Velocity smoke tests, Hangar publish workflow.

Each phase requires a `docs/product/<feature>.md` contract before code.
