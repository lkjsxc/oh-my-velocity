# Capability Map

## Current Capabilities

- Network join and leave messages.
- Server-list MOTD.
- Native tab-list presentation.
- `/hub` command.
- Reload.
- Scheduled and manual restart.

## Next Capabilities

- More precise config diagnostics.
- More pure planner coverage.
- Real clean-proxy smoke automation if a Velocity harness is introduced.

## Later Capabilities

- Cached tab-list dirty updates for large networks.
- Persisted first-seen player store if operators request it.

## Research-Only

- Name-tag, team, collision, or ping-display features that need a reliable
  self-contained implementation path before becoming active product behavior.
