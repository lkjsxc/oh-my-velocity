# Current Focus

The current focus is a self-contained, docs-first plugin.

## Active Work

- Remove third-party plugin runtime requirements.
- Keep native tab-list behavior that Velocity supports directly.
- Replace legacy config compatibility with one current schema.
- Move IO and runtime effects out of domain code.
- Strengthen verification so dependency drift is caught automatically.

## Stop Condition

This focus is complete when Docker Compose verification passes, forbidden
dependency scans are clean, and active docs match implementation behavior.
