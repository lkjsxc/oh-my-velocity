# Root Layout Contract

## Required Paths

- `docs/`, `scripts/`, `src/main/java/`, `src/main/resources/`
- `src/test/java/`, `build.gradle.kts`, `settings.gradle.kts`
- `gradlew`, `gradlew.bat`, `gradle/wrapper/`
- `docker-compose.verify.yml`

## Rules

1. Product contracts live under `docs/`.
2. Verification scripts live under `scripts/`.
3. Default config lives under `src/main/resources/`.
